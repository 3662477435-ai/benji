# CanopyValue

Commercial-grade static web app for urban street tree ecosystem value assessment.

## What is included

- Multi-city environmental parameter selection
- Tree species and street layout visualization
- Ecosystem value, risk, suitability, ROI, and biodiversity results
- Scheme comparison and share links
- PDF report generation
- Local vendored frontend dependencies for reliable GitHub Pages delivery

## Deployment

The site deploys through GitHub Pages from the `main` branch using `.github/workflows/pages.yml`.
